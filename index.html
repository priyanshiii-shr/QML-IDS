Project Overview & Technology

Academic QML exploration using scikit-learn & PennyLane frameworks

Abstract

High-dimensional packet classification poses critical runtime limits in energy-constrained IoT networks. This application compares classical tree ensemble networks (Random Forest Baseline) with a Variational Quantum Classifier (VQC) using a qubit mapping representation on scaled PCA feature boundaries.

Through Angle Encoding features into RY rotation nodes across a Strongly Entangling multi-layer ring topology, we simulate expectation predictions optimized by gradient descent via PyTorch backpropagation structures.

VQC Hardware Schema

🔌
Feature Map

🔗

📈

Adam gradient step (Parameter-shift rule)

Deployment Details

This web dashboard is deployed as a serverless static site on Vercel, integrating a Python serverless function microservice to host scikit-learn models natively in the cloud.

Development Stack

  • ⚛️ PennyLane 0.44 (Quantum Circuit simulation engine)
  • 🔥 PyTorch 2.x (Automatic differentiation backend)
  • 🌲 scikit-learn 1.3+ (Random Forest + scaling parameters)
  • 🐍 FastAPI (API routing middleware)
  • 📊 Chart.js (GPU-accelerated interactive web charts)

Research Limitations

  • Runs on standard classical state-vector simulators (no real quantum computers used in production demo).
  • QML circuit requires PCA feature reduction down to 8 dimensions to maintain reasonable memory usage (scales exponentially O(2^N)).
